# 배너 자동화 시스템 - 시작 가이드

## 전체 구조

```
웹 요청 페이지 → 백엔드 서버 → Figma 플러그인 → PNG 다운로드
```

---

## STEP 1. Node.js 설치

서버 실행에 필요합니다. (이미 설치됐으면 건너뛰세요)

1. https://nodejs.org 접속
2. **LTS 버전** 다운로드 및 설치
3. 터미널에서 확인:
   ```
   node -v
   ```
   버전 숫자가 나오면 성공

---

## STEP 2. 서버 설치 및 실행

터미널(Terminal.app)을 열고 아래 명령어를 순서대로 입력하세요:

```bash
cd /Users/yuchaewon/banner-automation/server
npm install
npm start
```

성공하면 이런 메시지가 나옵니다:
```
배너 자동화 서버 시작!
웹 요청 페이지: http://localhost:3000
```

> 서버는 배너 작업하는 동안 계속 켜두세요.

---

## STEP 3. 웹 요청 페이지 열기

브라우저에서: **http://localhost:3000**

---

## STEP 4. Figma 플러그인 등록

1. Figma 앱(데스크탑) 실행
2. 배너 템플릿 파일 열기
3. 메뉴: **Plugins → Development → Import plugin from manifest...**
4. 파일 선택: `/Users/yuchaewon/banner-automation/figma-plugin/manifest.json`

---

## STEP 5. 플러그인 실행

1. Figma에서 **Plugins → Development → 배너 자동화** 클릭
2. 플러그인 창이 열리고 "서버 대기 중..." 상태가 되면 준비 완료

---

## STEP 6. 배너 요청하기

1. 브라우저에서 http://localhost:3000 열기
2. 타이틀, 본문, 하이라이트 텍스트 입력
3. **배너 요청하기** 버튼 클릭
4. Figma가 자동으로 레이어를 업데이트하고 PNG를 다운로드 폴더에 저장

---

## 주의사항

- 서버(터미널)와 Figma 플러그인이 **둘 다 실행 중**이어야 합니다
- Figma에서 `title`, `txt_1`, `txt_2` 레이어가 있는 프레임이 현재 페이지에 있어야 합니다
- PNG는 Mac 기본 **다운로드 폴더**에 저장됩니다
- 파일명: `banner_날짜_시간.png` (예: `banner_20260519_143022.png`)

---

## 문제 해결

| 증상 | 해결 방법 |
|------|-----------|
| 서버 연결 실패 | 터미널에서 `npm start` 실행 확인 |
| 레이어를 못 찾음 | Figma 레이어 이름이 `title`, `txt_1`, `txt_2` 인지 확인 |
| 다운로드 안 됨 | 브라우저 팝업 차단 해제 확인 |
| 폰트 오류 | Figma에서 해당 폰트가 설치되어 있는지 확인 |
